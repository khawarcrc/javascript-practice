Reverse a String

Explanation for first function

  • This is a for loop that will iterate through the characters of the input string in reverse order.
  • `let i = str.length - 1;`: This initializes a loop control variable `i` to the index of the last character in the input string. `str.length` gives you the length of the string, and by subtracting 1, we start from the last character (JavaScript arrays and strings are zero-indexed, so the last character is at `str.length - 1`).
  • `i >= 0;`: This is the condition that controls how long the loop runs. It continues to run as long as `i` is greater than or equal to 0.
  • `i--`: This is the decrement statement, which means that in each iteration, `i` is reduced by 1.
